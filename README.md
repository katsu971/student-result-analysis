# Student Result Analysis (Python)

## Overview
This project performs exploratory data analysis on student academic performance data.
The analysis focuses on understanding score distributions and the impact of demographic
and background factors on student results.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Gender-wise performance comparison
- Subject-wise score distribution (Math, Reading, Writing)
- Grouped analysis based on parental education level
- Basic aggregation using Pandas groupby

## How to Run
1. Clone the repository
2. Install required libraries:
   pip install pandas matplotlib seaborn
3. Open `student_result_analysis.ipynb`
4. Run all cells from top to bottom
